HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used for the structure of the website, it is made up of elements that perform distinct notes.

CSS

  • CSS determines the styles of a webpage in order to make a website visually appealing.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates howmuch space we want around the content inside an element.

Git

  • Git is a version control system.
  • Github is a cloud based service that hosts our repositories, which is where the project code will live.
  • Github stores files of code, info about code and version history for all our files
  • Github allows multiple developers to work on the same project via branches, without affecting the main branch.
  • git status: checks what branch we are currently working on.
  • git checkout -b branch-name: creates a new branch and switches to the newly created branch.
  • git add -A command will commit all files in the working branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a high level dynamic programming language that allows users to interact with the website.
  • Conditional statements are code structures used to test if an expression returns true or false, like an if else statement.
  • Functions are blocks of code that executes a specific task only when we call it.
  • An array us a single variable that is used to hold and store a group of related data, which can hold multiple different data types; ex strings, booleans, integars, or chars.
  • A loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code of block to be executed over and over. If statements are crucial to interrupt the control flow.
